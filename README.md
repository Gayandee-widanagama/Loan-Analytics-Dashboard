# Loan Analytics Dashboard

## Project Overview
This project provides a comprehensive analysis of loan performance, customer behavior, and risk factors. It bridges the gap between raw data preparation using **SQL** and interactive storytelling using **Power BI**.

## Tools Used
* **SQL:** Data cleaning, transformation, and handling backend logic.
* **Power BI:** Data modeling, DAX measures, and interactive visualization.

## Key Insights
* **Urban Dominance:** Highest loan demand originates from urban areas.
* **Education Impact:** Graduates show a significantly higher loan approval rate.
* **Risk Mapping:** Relationship between income levels and loan amounts to identify safe borrower segments.

## How to View
1. Download the `Loan_Analytics.pbix` file.
2. Open it using Power BI Desktop.
